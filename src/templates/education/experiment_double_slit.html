实验原理

双缝干涉是波动光学的经典实验，由托马斯·杨于1801年首次进行。当相干光通过两个狭缝时，会在屏幕上形成明暗相间的干涉条纹。

干涉条纹公式

条纹间距：Δy = λL/d

明条纹位置：y = mλL/d (m = 0, ±1, ±2, ...)

暗条纹位置：y = (m+1/2)λL/d (m = 0, ±1, ±2, ...)

光强分布：I = I₀cos²(πdy/λL) · sinc²(πay/λL)

其中，λ为波长，L为屏距，d为双缝间距，a为缝宽

实验步骤

  1. 调整波长λ，观察不同颜色光的干涉条纹变化。
  2. 调整双缝间距d，观察条纹间距的变化。
  3. 调整屏幕距离L，观察条纹宽度的变化。
  4. 调整缝宽a，观察单缝衍射对双缝干涉的调制作用。

物理意义

双缝干涉实验证明了光的波动性，是量子力学中波粒二象性的重要基础。条纹间距与波长成正比，与缝间距成反比，这反映了波的干涉规律。
